# moot-code
Code for baseline results from the MOOT repo

To install:

```bash
# install python, code support for sh utils (make, gawk, grap, etc)
mkdir someDir
cd someDir
git clone http://github.com/timm/moot
git clone http://github.com/timm/moot-code
cd moot-code
make  eg0
```
